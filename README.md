# TikTok Downloader - PM Code with AI

## 中文版

### 简介

TikTok Downloader 是一款由 PM Code with AI 开发的浏览器扩展，让您能够轻松下载抖音和 TikTok 视频。

### 功能特点

- 一键下载抖音视频
- 一键下载 TikTok 视频
- 简单易用的界面
- 支持多种语言（包括中文、英文、德文、法文、西班牙文等13种语言）

### 安装方法

1. 访问 [Chrome 网上应用店](https://chromewebstore.google.com/detail/cacapadmnfkadndppbealidbikccgojl?utm_source=item-share-cb) 下载安装扩展
2. 在浏览器中打开扩展
3. 访问抖音或 TikTok 网站
4. 在视频页面点击扩展图标即可下载视频

### 支持的网站

- 抖音 (*.douyin.com)
- TikTok (*.tiktok.com)

## English Version

### Introduction

TikTok Downloader is a browser extension developed by PM Code with AI that allows you to easily download videos from Douyin and TikTok.

### Features

- Download Douyin videos with one click
- Download TikTok videos with one click
- Simple and easy-to-use interface
- Support for multiple languages (including English, Chinese, German, French, Spanish, and 8 other languages)

### Installation

1. Visit the [Chrome Web Store](https://chromewebstore.google.com/detail/cacapadmnfkadndppbealidbikccgojl?utm_source=item-share-cb) to download and install the extension
2. Open the extension in your browser
3. Visit Douyin or TikTok websites
4. Click on the extension icon on the video page to download the video

### Supported Websites

- Douyin (*.douyin.com)
- TikTok (*.tiktok.com)
